webhook

URL do Webhook Hotmart

Cole esta URL no painel da Hotmart

Sua URL:

https://SEU-SITE.netlify.app/.netlify/functions/hotmart-webhook

Hotmart → Ferramentas → Webhooks → Nova integração → cole esta URL → selecione PURCHASE_APPROVED

Passo a passo

1

Deploy no Netlify

Extraia o ZIP e arraste a pasta em app.netlify.com. Grátis.

2

Configure o webhook na Hotmart

Copie a URL acima e cole no painel da Hotmart.

3

Banco de dados Supabase (opcional)

Para liberar acesso automático. Grátis em supabase.com.

Código webhook (netlify/functions/hotmart-webhook.js)

exports.handler = async (event) => { const body = JSON.parse(event.body || '{}'); const email = body?.data?.buyer?.email; const productId = body?.data?.product?.id; const evt = body?.event; if (evt === 'PURCHASE_APPROVED' && email) { // Liberar acesso no Supabase // 1. Salvar membro (INSERT ou UPDATE em members) // 2. Liberar produto pelo hotmart_id // 3. Liberar bônus vinculados automaticamente console.log('Acesso liberado:', email, 'produto:', productId); } return { statusCode: 200, body: 'ok' }; };
database

Banco de Dados Supabase

Cole as credenciais do seu projeto em supabase.com

Encontre em: Settings → API → Project URL

Encontre em: Settings → API → Project API keys → anon / public

⚠️ Importante: Após salvar aqui, adicione também as variáveis SUPABASE_URL e SUPABASE_ANON_KEY nas variáveis de ambiente do seu site no painel da Netlify (Site settings → Environment variables). Isso é necessário para o webhook funcionar no servidor.
